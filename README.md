## ✨ Fitur Utama
*   **Modern Premium Aesthetic**: Dibangun dengan nuansa *Dark Mode* dipadukan aksen neon (*Cyan* & *Purple*) yang memberikan kesan elegan.
*   **Glassmorphism UI**: Penggunaan panel kaca semi-transparan kekinian tanpa membebani performa browser.
*   **100% Vanilla Power**: Tidak mengandalkan *framework* CSS seperti Tailwind atau Bootstrap. Semua animasi tingkat lanjut (`fadeOut`, `slideUp`), *responsive grid*, dan gaya *modern* diracik manual (HTML, CSS, JS murni).
*   **Micro-Interactions**: Dihiasi background animasi geometrik *canvas* yang interaktif interaktif, serta transisi ketik (typer) dan *smooth scroll*.

---

## 🛠️ Tech Stack & Tools
Website ini dimaksimalkan untuk performa cepat karena berfokus pada teknologi *frontend* yang *native*:
*   **Struktur**: HTML5 (Semantic Markup)
*   **Gaya (Styling)**: Vanilla CSS3 (Custom Properties & Flexbox/Grid layouts)
*   **Fungsionalitas**: Vanilla JavaScript (ES6+), Interaction Observer API
*   **Font & Ikon**: Google Fonts (Outfit & Inter), FontAwesome (v6)

---

## 📁 Struktur Kode
*   `index.html` — Halaman tunggal (*Single Page Application*) yang interaktif.
*   `style.css` — Penyimpan sistem gaya, token warna, responsivitas (ukuran *mobile* vs *desktop*), dan keyframes animasi.
*   `script.js` — Logika *scroll-interceptor*, fungsionalitas hamburger menu (mobile), dan integrasi library pihak ketiga.
*   `/assets` — Berisi gambar, screenshot aplikasi (*mockups*), dan dokumen (seperti file PDF).
